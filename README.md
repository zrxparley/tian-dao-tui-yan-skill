# 天道推演 (Tian-Dao)

> 天神视角的沙盘推演系统，具备八维推演能力，能推演未来各种可能结果，并持续自我进化提升。

## 功能

- 🔮 **八维推演** - 因果链/博弈方/变量/分支点/概率/时间线/蝴蝶效应/终局
- 🧠 **毛选思维** - 矛盾论、实践论、统一战线、持久战、群众路线等9大模型
- 📊 **芒格思维** - 第一性原理、逆向思维、二阶思维、事前验尸、反脆弱
- 🔄 **自我进化** - 每次推演后自动记录，持续校准提升准确率

## 使用方式

当你想进行推演时，说：
- "天道推演：..."
- "帮我分析下..."
- "推演一下..."
- "如果...会怎样"
- "预测一下..."

## 核心原则

1. 不预测唯一未来 - 列出所有可能
2. 永远有B计划 - 确保能承受最坏结果
3. 概率思维 - 不是"会怎样"，而是"各有多少可能"
4. 持续校准 - 每次重大事件后复盘
5. 抓住主要矛盾 - 复杂局面中找到核心杠杆点

## 安装

将 `tian-dao/` 目录放入你的 Agent Skills 目录即可。
