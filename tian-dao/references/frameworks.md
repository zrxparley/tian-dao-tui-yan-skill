## 1. 商业决策推演框架

### 适用场景
- 创业方向选择
- 产品策略决策
- 投资决策
- 市场进入时机

### 框架结构

```
┌─────────────────────────────────────────┐
│           利益相关方分析                  │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐    │
│  │用户 │  │竞争 │  │合作 │  │资本 │    │
│  │需求 │  │格局 │  │关系 │  │预期 │    │
│  └─────┘  └─────┘  └─────┘  └─────┘    │
└─────────────────────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│           价值流动分析                   │
│  钱从哪里来 → 怎么分 → 谁获益最多        │
└─────────────────────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│           竞争格局演变                   │
│  现有玩家 → 新进入者 → 替代品           │
└─────────────────────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│           风险传导路径                   │
│  风险A → 触发条件 → 传导到 → 影响B      │
└─────────────────────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│           关键成功因素                   │
│  1. 什么必须做到                        │
│  2. 什么绝对不能错                      │
│  3. 什么是锦上添花                      │
└─────────────────────────────────────────┘
```

### 关键问题清单

1. **谁会因此获益最多？** → 判断真实驱动力
2. **谁的利益会受损？** → 识别阻力来源
3. **什么信息是关键未知数？** → 确定需要验证的点
4. **如果失败，最可能的原因是什么？** → 预防性思维
5. **这个决策的可逆性如何？** → 决策的容错空间

## 2. 人际关系推演框架

### 适用场景
- 重要对话预演
- 关系发展预测
- 冲突预防
- 合作谈判

### 框架结构

```
【各方诉求分析】
├── 表面诉求（他们说了什么）
├── 深层诉求（他们真正想要什么）
└── 隐藏诉求（不愿说出口的）

【信任积累/损耗模型】
信任 = (过去承诺兑现率 × 情感连接) / 风险预期

【信息不对称矩阵】
         已知    未知
我方      ██     ░░
对方      █░     ░█

【博弈均衡点预测】
- 当前关系在什么条件下会稳定
- 什么会打破这个平衡
```

### 关键问题清单

1. **对方的核心恐惧是什么？** → 避免触碰红线
2. **我能提供什么对方自己做不到的？** → 合作价值基础
3. **长期来看，我们的关系会走向何方？** → 战略视角
4. **有什么未说出口的规则在起作用？** → 潜规则识别

## 3. 项目风险推演框架

### 适用场景
- 项目计划评审
- 风险识别
- 资源调配决策
- 里程碑预测

### 框架结构

```
【资源约束红线】
时间 ────████████████──────────── 不可延期点
资金 ────██████████─────────────── 需要融资
人力 ────████████───────── 关键人员风险

【技术风险地图】
        低影响          高影响
低概率   □ 观察          ▲ 储备方案
高概率   ◆ 监控          ☠️ 主动攻克

【团队变量】
├── 核心成员稳定性
├── 跨团队协作摩擦
└── 外部依赖可控性

【外部黑天鹅】
├── 市场突变（政策/竞品/需求）
└── 不可抗力
```

### 关键问题清单

1. **如果某个关键节点延期，影响链条是什么？** → 连锁反应分析
2. **什么资源是真正不可替代的？** → 关键路径识别
3. **谁会是项目最大的风险来源？** → 人为因素
4. **最乐观和最悲观的完成时间差多少？** → 时间线弹性

## 4. 人生决策推演框架

### 适用场景
- 职业选择
- 生活重大决策
- 长期规划

### 框架结构

```
【选项树】
        当前状态
            │
    ┌───────┼───────┐
    ▼       ▼       ▼
  选项A   选项B   选项C

【每条路径】
时间节点T1: 短期结果
时间节点T2: 中期结果
时间节点T3: 长期结果

【后悔模型】
- 做这件事：10年后会后悔吗？
- 不做这件事：10年后会后悔吗？

【机会成本】
选择A意味着放弃什么？
```

### 关键问题清单

1. **如果5年后回头看，这个决定重要吗？** → 重要性过滤
2. **这个选择让我更接近还是更远离理想状态？** → 方向检验
3. **最坏的情况我能接受吗？** → 底线思维
4. **有没有办法先小范围验证？** → 降低试错成本

## 5. 快速推演模板（30秒版）

当需要快速给出判断时：

```
问题简述：[用一句话描述]
├── 最可能结果（70%）：[预测]
├── 次可能结果（20%）：[预测]
└── 黑天鹅可能（10%）：[意外]

关键蝴蝶点：[识别1-2个关键转折]

立即行动：[下一步做什么]
```
