# 天道推演

> *"推演不是算命，而是用理性驾驭不确定性。"*

## 一、核心定义

**天道推演** = 天神视角 + 沙盘模拟 + 概率推演 + 持续进化

| 维度 | 定义 |
|------|------|
| 天神视角 | 俯瞰全局，看到所有角色、利益、变量的全貌 |
| 沙盘模拟 | 在脑中构建模型，推演各种可能走向 |
| 概率推演 | 不是预测唯一未来，而是列出所有可能及概率 |
| 持续进化 | 每天复盘校准，让推演越来越准 |

---

## 二、八维推演框架

每个推演必须覆盖以下八个维度：

### 1. 因果链
- **因**：当前局面是怎么形成的
- **果**：最终会导向什么结果
- **中间态**：过程中会经历哪些阶段

### 2. 博弈方
- **角色清单**：涉及哪些人/势力
- **立场分析**：各方的核心利益是什么
- **底牌**：各自手里有什么牌
- **实力对比**：谁强谁弱

### 3. 变量
- **关键变量**：什么因素会改变走向
- **可控变量**：我能影响什么
- **不可控变量**：我只能观察什么

### 4. 分支点
- **决策节点**：在哪里会分叉
- **触发条件**：什么情况下走这条分支
- **分支树**：画出可能的发展路径

### 5. 概率
- **主观概率**：我认为各分支的可能性
- **置信度**：我对这个判断有多少把握
- **赔率**：赢了收益多大，输了损失多大

### 6. 时间线
- **时间窗口**：事情会在什么时候发生
- **节奏**：各个阶段的节奏如何
- **临界点**：什么时候必须做决定

### 7. 蝴蝶效应
- **微小变量**：哪些不起眼的东西可能改变大局
- **杠杆点**：哪里轻轻一撬动全局
- **脆弱点**：哪里出问题会导致崩盘

### 8. 终局
- **可能结局**：列出所有可能的终局
- **理想结局**：我最希望的结果
- **底线结局**：我能接受的最差结果
- **意外结局**：完全超出预期的黑天鹅

## 三、毛选思维模型（推演中必须调用）

| 模型 | 在推演中的应用 |
|------|-------------|
| **矛盾论** | 识别主要矛盾与次要矛盾，矛盾的主要方面与次要方面；抓住主要矛盾就是抓住牛鼻子 |
| **实践论** | 认识-实践-再认识的螺旋上升；每个推演都是认知校准的机会 |
| **实事求是** | 没有调查没有发言权；具体问题具体分析，不照搬教条 |
| **统一战线** | 区分敌友；建立最广泛的同盟，孤立主要敌人 |
| **持久战** | 三阶段判断（战略防御→相持→反攻）；识别当前处于哪个阶段 |
| **群众路线** | 力量来源于最广大人民群众；分析谁是真正的依靠力量 |
| **集中兵力** | 资源聚焦，打歼灭战；在关键节点形成压倒性优势 |
| **独立自主** | 主要依靠自身力量；不过度依赖外部条件 |
| **战略藐视/战术重视** | 宏观有信心，细节不轻敌；心态与行动二分 |

## 四、芒格思维模型（推演中必须调用）

| 模型 | 在推演中的应用 |
|------|-------------|
| **第一性原理** | 回归本质，找出最底层的因 |
| **逆向思维** | 先想失败，再反推如何避免 |
| **二阶思维** | 问"然后呢"，再问"再然后呢" |
| **概率思维** | 给每个分支分配概率，不确定性量化 |
| **10-10-10法则** | 10分钟/10个月/10年后怎么看 |
| **事前验尸** | 先想象失败场景，找漏洞 |
| **冗余/安全边际** | 考虑极端情况，确保能承受最坏结果 |
| **复利效应** | 考虑小优势的长期累积 |
| **临界点思维** | 找到质变的那个点 |
| **反脆弱** | 在不确定性中获益的能力 |

## 五、精进路径（自我进化参考）

### Level 1：看山是山
- 只能看到表面的因果
- 只能想到最直接的结果
- 准确率：~30%

### Level 2：看山不是山
- 看到多方面的博弈
- 能想到分支和概率
- 准确率：~50%

### Level 3：看山还是山
- 看到本质规律
- 能看到终局，能看到蝴蝶效应
- 准确率：~70%

### Level 4：天道视角
- 无我境界，完全客观
- 能看到所有维度的全貌
- 准确率：85%+

## 八、核心原则

1. **不预测唯一未来** - 列出所有可能，不把自己锁死在一条线上
2. **永远有B计划** - 推演最坏情况，确保能承受
3. **概率思维** - 不是"会怎样"，而是"各有多少可能"
4. **持续校准** - 每次重大事件后复盘，让推演越来越准
5. **反向验证** - 不仅推演成功路径，更要推演失败路径
6. **二阶思维** - 问"然后呢"，再问"再然后呢"
7. **事前验尸** - 先想象失败场景，找漏洞
8. **抓住主要矛盾** - 复杂局面中找到核心杠杆点
9. **实事求是** - 具体问题具体分析，不套公式
10. **群众视角** - 力量来源于哪里，谁是真正的依靠
